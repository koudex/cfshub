Frequently Asked Questions

How do I access the gallery images?
You can access gallery images by navigating to the Gallery page. Images are organized into collections that you can browse. Click on any image to view it in full size, and use the download button to save it to your device.
How can I get admin access to post announcements?
Admin access is currently limited to class officers and teachers. If you need to post an announcement, please contact one of the class officers who can add it for you. The admin passcode is changed regularly for security.
Is my data secure on this website?
Yes, we take data security seriously. The website doesn't store personal information beyond what's necessary for functionality. Images are stored securely in Google Drive, and all communications are encrypted.
The website isn't loading properly on my device. What should I do?
First, try refreshing the page. If that doesn't work, clear your browser cache and try again. The website is designed to work on all modern browsers and devices. If problems persist, please contact us using the information below.
How often is the website updated with new content?
The website is updated regularly with new announcements, gallery images, and resources. Announcements are posted as needed by teachers and class officers. Gallery images are updated whenever new photos are taken during class activities.
